Research Interests

I am interested in data driven system development in computer vision. In specific, my works span on applications about computer vision-based assistive technology, sensor data to monitor human health status and apply machine learning in designing automated speech recognition applications.

Projects


Vision-based Realtime Accessibility Issues Detection

The project aims at using machine vision to detect accessibility issues on sidewalks for helping people with visually-impairment. A prior-based framework is designed to accelerate the computer vision detection process in embedded devices, which uses a cooperation of human and machine.

Demo







Vision-based Accurate Bus Stop Localization

We target to detect and localize bus stop accurately to improve the accessibility of bus station for blind riders. A bus stop detection system is developed and implemented completely in a smartphone platform.

Poster Demo







Weed Infestation Identification

We have designed to identify weed in the crop fields in smart-home platform. We developed both iOS and android application and also have backend intelligence to reduce the cost and time of identifying invasive weeds
